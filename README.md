# ShadowsOfTheCarpathians

A 2D pixel-art action RPG inspired by Romanian folklore and mythology.

## About

Shadows of the Carpathians is a game developed with LibGDX and Kotlin. Explore dark forests, abandoned villages and ancient ruins while fighting creatures inspired by Romanian legends.

## Features

- Pixel-art graphics
- Top-down exploration
- Melee combat
- Romanian folklore-inspired enemies
- Multiple weapons and upgrades (planned)
